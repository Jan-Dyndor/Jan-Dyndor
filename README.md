## 🏗 Life_ETL – Automated Databricks ETL Pipeline
🔗 https://github.com/Jan-Dyndor/Life_ETL  
🌐 Live demo: https://lifeetl-tumyuyaf3tuxwfok8ssf5c.streamlit.app/

End-to-end **automated ETL pipeline built on Databricks** that processes real financial data from the **Narodowy Bank Polski (NBP) public API**.

The pipeline runs automatically once per day (currently due to inactivity daily updates stopped) and sends an **email notification after each successful execution**, simulating a production-style data pipeline.

**Key Features**

- Automated daily ETL pipeline
- Real financial data ingestion from the NBP API
- Databricks job orchestration
- Email notification after successful execution
- Medallion architecture (Bronze / Silver / Gold layers)
- Streamlit dashboard for data inspection
- Testing and logging

**Technology Stack**

- Python
- Databricks
- PySpark
- NBP Public API
- Streamlit
- PyTest
- Loguru
- Docker

## 📊 Text Processing API with Observability
🔗 https://github.com/Jan-Dyndor/Text_API_and_GRAFANA_observability

A **production-style FastAPI backend** providing deterministic text-processing utilities with built-in **observability and monitoring**.

The project focuses on **backend engineering fundamentals**: clean architecture, testing, containerization, and monitoring.

**Key Features**

- FastAPI REST API
- Text cleaning and analysis endpoints
- SQLite persistence
- Automated testing (pytest)
- Containerized with Docker
- Observability stack with Prometheus, Loki, and Grafana
- Metrics endpoint for Prometheus scraping

**Observability Stack**

- Prometheus – metrics collection
- Loki – centralized logging
- Grafana – dashboards and monitoring

**Available Services**

| Service | URL |
|------|------|
| API Docs | http://localhost:8000/docs |
| Health Check | http://localhost:8000/health |
| Prometheus | http://localhost:9090 |
| Grafana | http://localhost:3030 |
